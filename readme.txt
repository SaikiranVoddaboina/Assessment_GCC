// For Open Weather API progarm
1.We  have to install libcurl (e.g., sudo apt install libcurl4-openssl-dev on Ubuntu).
2.Now, Replace "your_openweather_api_key" with your actual OpenWeather API key.
3.We can compile the code using gcc -o weather_app weather_app.c -lcurl.
4.Final step run the application with ./weather_app.
